# 个人简历 Personal Resume

这是一个静态个人简历网站，用于展示教育背景、技术能力、实习经历、项目方向与联系方式。

## 在线访问

公开网站地址：

https://skychutt.github.io/Personal-Resume/

这个地址部署在 GitHub Pages 上，别人可以直接访问，不依赖你的电脑是否开机。

## 本地预览

本地预览只适合在自己的电脑上查看或修改项目。

方式一：直接双击打开 `index.html`。

方式二：在当前目录启动本地静态服务：

```bash
python -m http.server 8765
```

然后在自己的电脑上访问：

```text
http://127.0.0.1:8765/
```

注意：`127.0.0.1` 是本机地址，只能在运行服务的这台电脑上使用。其他人不能通过这个地址访问你的简历网站；分享给别人时请使用上面的 GitHub Pages 在线地址。
